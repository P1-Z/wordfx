# WordFX soft sound library

These WAV files are original procedural sounds generated by
`tools/generate-soft-sounds.js`. They use no third-party samples and can be
regenerated deterministically with:

```powershell
node tools/generate-soft-sounds.js
```

The bank is intentionally quiet and short: felt taps, warm sine chimes,
filtered noise, smooth tape texture, and small pitch gestures. File amplitude is
limited at generation time so rapid interactions remain soft.
